# obsidian-plugin-styling

Style Obsidian plugins with CSS. Covers Obsidian CSS variables, dynamic stylesheet injection, `styles.css` loading and scoping, dark/light mode adaptation, theme-specific selectors, conflict avoidance, mobile styling, and animation best practices.

## Usage

This skill is triggered when you ask about:

- Obsidian plugin CSS / styling / themes
- Dark or light mode support for plugins
- Mobile layout for plugins
- Stylesheet injection or `styles.css` patterns
