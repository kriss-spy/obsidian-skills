# obsidian-plugin-bootstrap

Bootstrap a new Obsidian plugin from zero. Covers dev environment setup, manifest.json, project anatomy, plugin lifecycle, hot reload workflow, mobile constraints, and building for release.

## What This Skill Covers

- Setting up Node.js, esbuild, TypeScript, and the build pipeline
- Writing a correct `manifest.json` with proper ID, versioning, and compatibility
- Structuring a maintainable plugin project (`src/` folder, settings, commands, views)
- Understanding `onload()`, `onunload()`, and `onUserEnable()`
- Configuring hot reload for fast iteration
- Mobile vs desktop constraints and the `isDesktopOnly` flag
- Building production artifacts (`main.js`, `manifest.json`, `styles.css`)

## When to Use

Use this skill when scaffolding a new plugin or onboarding a developer to the Obsidian plugin ecosystem.
