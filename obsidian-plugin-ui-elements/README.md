# obsidian-plugin-ui-elements

A focused OpenCode skill for building UI elements in Obsidian plugins.

## Contents

- `SKILL.md` — Main skill documentation covering ribbon icons, status bar items, HTML helpers, icons, and RTL support.
- `references/` — Thin reference docs extracted from official Obsidian documentation.

## What It Covers

- `addRibbonIcon()` — adding actions to the left ribbon
- `addStatusBarItem()` — desktop-only status bar blocks
- `createEl()`, `createDiv()`, `createSpan()`, `createFragment()` — DOM element helpers
- Built-in Lucide icons via `setIcon()` and custom SVG via `addIcon()`
- Right-to-left (RTL) language support and accessibility patterns

## When to Use

Use this skill when you need to add, style, or manage UI components in an Obsidian plugin.
